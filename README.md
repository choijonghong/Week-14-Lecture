Week-14-Lecture

### 적대와 경쟁은 어떻게 AI에게 상상력을 가르쳤는가? – GAN의 탄생
* 논문명: Generative adersarial Nets
* "복잡한 수학 계산 대신, '만드는 놈'과 '감시하는 놈'을 싸우게 시켰더니 놀라울 정도로 진짜 같은 결과물이 나왔다."

### 논문배경
* 딥러닝은 무엇인지 구분하는 기술은 뛰어났지만, 새로 만들어내는 기술은 부족했다.
* 이미지를 생성하려면 픽셀들의 복잡한 확률 관계를 계산해야 했고, 이는 너무 어려운 문제였다.
* 기존 방법은 정확히 계산할 수 없어 근사 기법에 의존했다.
* 그 결과 흐릿하고 현실감 없는 이미지가 만들어졌다.
* 생성 모델의 발전은 딥러닝의 다음 단계였지만, 해결책이 보이지 않았다.
* 이 한계를 넘기 위해 GAN은 ‘경쟁’이라는 새로운 학습 방식을 제안하며 등장했다.

### 기존 연구의 한계점
* RBM·DBM 등 이전 생성 모델은 복잡한 확률 계산(MCMC, Partition Function) 때문에 느리고 구현이 어려웠다.
* VAE는 이미지를 생성했지만 초점이 흐린 듯한 결과가 나오는 한계가 있었다.
* 명확한 확률 분포를 정의해야 해서 설계가 까다롭고 유연성이 떨어졌다.

### 논문 목표
* 복잡한 확률 추론을 배제하고 역전파 기반으로 학습 가능한 생성 모델을 제안한다.
* 두 신경망을 경쟁시키는 적대적 학습 구조(Minimax Game)를 도입한다.
* 생성 모델의 분포가 실제 데이터 분포와 동일해지도록 학습하는 방법을 확립한다.

### 논문내용(주요 혁신 및 방법론)  
* 이 부분은 **'위조지폐범(Counterfeiter)'**과 **'경찰(Police)'**의 비유로 완벽하게 설명됩니다.
* 두 명의 주인공
    * Generator (G, 생성자 = 위조지폐범):
      * 아무런 의미 없는 랜덤 숫자(노이즈, $z$)를 받아서, 진짜 같은 **가짜 데이터(위조지폐)**를 만들어냅니다.
      * 목표: 경찰(D)이 이 지폐를 진짜라고 착각하게 만드는 것.

    * Discriminator (D, 판별자 = 경찰):
      * 사진을 보고 이것이 **'실제 데이터'**인지, **'G가 만든 가짜'**인지 감별합니다.
      * 목표: 가짜를 기가 막히게 잡아내는 것.

* 학습과정(경쟁의 시작)
    * D의 turn: 진짜와 가짜를 보여주며 “이게 진짜인지 맞혀봐!” 하고 훈련한다.
    * G의 turn: 들킨 가짜를 보며 “이렇게 만들면 안 걸리겠군” 하고 더 자연스럽게 바꾼다.
    * 반복: 서로 속이고 잡는 과정을 계속하며 둘 다 점점 더 똑똑해진다.

* 최적의 상태 (Global Optimality)
    * 논문에서는 수학적으로 증명합니다. "이 경쟁을 끝까지 가면 어떻게 될까?"
    * 결과: 위조지폐범(G)이 너무 완벽해져서 진짜 데이터와 똑같은 데이터를 만들어냅니다.
    * D의 항복: 경찰(D)은 이제 들어오는 데이터가 진짜인지 가짜인지 구별할 수 없어서, 확률을 50 대 50 으로 찍게 됩니다.
    * 이것이 학습 완료 시점입니다.

### 기여도 및 결과
* GAN은 경쟁을 활용한 새로운 학습 패러다임(Adversarial Training) 을 제시했다.
* 기존보다 선명하고 현실적인 샘플을 생성할 수 있었다.
* 노이즈를 넣으면 한 번에 샘플을 생성할 수 있어 빠르고 효율적이었다.
* 이후 DCGAN, CycleGAN, StyleGAN 등 생성형 AI의 발전을 이끄는 기반 기술이 되었다.

### 감성컴퓨팅과 GAN
* 감성컴퓨팅은 감정을 이해하는 기술, GAN은 감정을 표현하고 생성하는 기술이다.
* GAN은 감정이 담긴 얼굴·음성·표현을 만들 수 있어 감성 상호작용에 기여한다.
* 감성 데이터 부족 문제를 가상 데이터 생성으로 보완할 수 있다.
* 결과적으로 GAN은 감성컴퓨팅을 분석 중심에서 창조 중심으로 확장시키는 역할을 한다.
